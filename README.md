# Personal Portfolio Website

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript, featuring a clean neumorphic design.

## Features

- **Responsive Design**: Optimized for all device sizes
- **Neumorphic UI**: Modern shadowing effects with a clean interface
- **Interactive Elements**: Hover effects, animations, and smooth transitions
- **Modular Structure**: Clean component-based organization

## Technologies Used

- HTML5
- CSS3 (with custom properties)
- Tailwind CSS
- JavaScript
- Phosphor Icons
- Google Fonts (JetBrains Mono, Inter)

## Design Specifications

- **Color Palette**:
  - Primary Background: Deep Space Gray (#2D3748)
  - Dark Background: #1A202C
  - Accent Color: Code Green (#48BB78)
  - Text: Light Gray (#E2E8F0)

- **Typography**:
  - Headings: JetBrains Mono (Bold)
  - Body Text: Inter (Regular)

- **Visual Effects**:
  - Neumorphic shadows
  - Smooth hover transitions
  - Animated skill tags with flowing patterns

## Sections

1. **Bio Card**: Personal introduction and quick links
2. **Skills Radar**: Visual representation of technical proficiencies
3. **Project Matrix**: Showcase of notable projects with links
4. **Experience Timeline**: Professional history displayed chronologically
5. **Contact Panel**: Contact form and alternative contact information

## Usage

1. Clone the repository
2. Open `index.html` in your browser
3. Customize content by editing the HTML files
4. Modify styles in the CSS section to match your preferences

## Customization

You can easily customize:
- Profile information
- Skills and proficiency levels
- Projects and their details
- Experience timeline
- Contact information
